1. 从D网打包
在 172.21.43.254 打包前后端镜像
```bash
> docker save qkos/qiko-plus/api:dev qkos/qiko-plus/web:dev  202407180930qiko071.tar
> gzip 202407180930qiko071.tar
```
2. 用光驱拷贝到B网
最终拷到 10.30.49.11 ： 
3. docker compose 文件位置： /var/opt/dify/docker/docker-compose.yml
4. 加载镜像：
```bash
> docker load -i 202407180930qiko071.tar.gz
```
5. 部署更新：docker-compose -f docker-compose.yml up -d

> ！！！注意：每次重启都使用 `docker-compose -f docker-compose.yml up -d` ！！！
