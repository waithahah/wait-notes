# 安装 pipx — 在隔离环境中安装并运行 Python 应用程序
[pipx github地址](https://github.com/pypa/pipx)

# 安装 poetry 
[poetry 安装说明](https://python-poetry.org/docs/)
- 安装脚本：`pipx install poetry`

# 添加清华源
`poetry source add tsinghua https://pypi.tuna.tsinghua.edu.cn/simple`

# 重新锁定依赖
`poetry lock`

# 使用 poetry 安装依赖
`poetry install`

# 使用 poetry 运行程序
`poetry run python -m flask db upgrade`

# 查看poetry配置信息
`poetry config --list`
