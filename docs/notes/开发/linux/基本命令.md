# 基础命令

1. 查看系统：

    ```sh
    cat /etc/os-release
    ```

    ```sh
    lscpu
    ```

2. 查看系统架构：uname -m

3. 压缩为 .tar.gz 格式（最常用）

    ```bash
        tar -czvf archive-name.tar.gz folder-name/
    ```

    参数说明：
        c: 创建压缩文件

        z: 使用 gzip 压缩

        v: 显示详细过程（可选）

        f: 指定文件名

        archive-name.tar.gz: 压缩后生成的文件名

        folder-name/: 要压缩的文件夹

4. 解压 .tar.gz 文件

    ```bash
        tar -xzvf archive-name.tar.gz
    ```

    说明：

        x: 表示解压

        z / j: 分别表示解压 .gz 和 .bz2

        v: 显示详细信息（可选）

        f: 指定文件名


5. 支持断点续传的命令，中断后只要重新执行即可

rsync -avh --progress --partial storage.tar.gz /data/bosfsDir/livebucket/qiko/
