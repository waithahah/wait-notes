# 适配昇腾NPU

> 目前测试过的硬件主要有 300Ipro 300Vpro

## 适配方法

### 方法一：mindnlp 0.4.1 + mindspore 2.4.1

适配步骤：
- 下载模型：git lfs install && git clone https://www.modelscope.cn/openai-mirror/whisper-large-v3-turbo.git
- 编码实现：实现在PolyVoice项目：PolyVoice/backend/app/core/model_runtime/model_providers/mindnlp_whisper_model.py
- 构建镜像：cd PolyVoice/backend && make build-asr-ascend
- 部署：cd docker-polyvoice && bash deploy.sh 后选择配置 125ascend.env

华为开放环境运行步骤 [https://xihe.mindspore.cn/training-projects]：
- uv创建虚拟环境安装依赖等

### 方法二：Whisper模型（TorchAir）-推理指导 [https://gitee.com/ascend/ModelZoo-PyTorch/blob/master/ACL_PyTorch/built-in/audio/whisper/README.md]


### 方法三：Whisper模型（MindIE / MindIE-Torch） 推理指导 [https://gitee.com/ascend/ModelZoo-PyTorch/blob/master/MindIE/MindIE-Torch/built-in/audio/Whisper/README.md]
