1. 进程 → 真并行 ✅（适合 CPU 密集（计算、模型推理）） 独立内存 
2. 线程 → 可并行（但 Python 受 GIL 限制）⚠️（更适合 IO 密集） 共享内存 ❗ Python 受 GIL 限制
3. 协程 → 单线程内调度（高并发 IO）✅ ✅ 单线程
✅ 无切换开销
✅ 极高 IO 并发
❗ 不能利用多核 大量 IO（HTTP / DB / Redis / LLM）
