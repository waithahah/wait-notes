# 消息队列流程

## redis stream

### 初始化消息队列

1. video.analyse.local
2. video.analyse.ftp
3. video.analyse.oss
4. video.extract.completed
5. video.analyse.result
6. audio.extract.completed
7. audio.recognize.completed

#### redis stream 相关命令

1. 创建消息队列命令

   ```shell
    xadd video.analyse.local * video_id 123
   ```

2. 创建消费者组命令

   ```shell
        xgroup create video.analyse.local video_analyse_local_nex_group $
   ```
3. 删除队列命令

   ```shell
        DEL video.analyse.local
   ```

### 初始化消费者组

1. video.analyse.local -> video_analyse_local_nex_group
2. video.analyse.ftp -> video_analyse_ftp_nex_group
3. video.analyse.oss -> video_analyse_oss_nex_group
4. video.extract.completed -> video_extract_completed_nex_group
5. video.analyse.result -> audio_extract_completed_nex_group
6. audio.extract.completed -> audio_recognize_completed_nex_model_group、audio_recognize_completed_nex_control_group、audio_recognize_completed_nex_named_group

#### 消费者组相关命令

1. 查看消费者组命令

   ```shell
        XINFO GROUPS stream:audio.extract.completed
   ```

2. xgroup setid
3. xgroup delconsumer
4. xgroup del
