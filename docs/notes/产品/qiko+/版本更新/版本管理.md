# 版本管理：
## 分支管理：
1. `main`分支：最近同步的dify最新版
2. `master`分支：qiko+最新稳定版，后续部署包将从这个分支产生
3. `dev`分支：研发分支
## tag管理
1. 每次版本发布后将生成对应的tag，管理所有已发布版本

# 互联网研发环境和正式环境更新步骤：
## 互联网研发环境[172.21.43.254](http://172.21.43.254/)
1. `dev`分支提交代码即可自动更新
## 互联网正式环境
1. bamboo上手动点击 `master`分支 的构建计划
2. 构建完成后，手动点击 `master`分支 的部署计划
3. 即可自动更新正式环境
