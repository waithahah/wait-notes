1. 获取智能体id
* 通过页面 f12 接口 GET:/console/api/apps?page=1&limit=30 获取
* 通过调用接口 GET:/console/api/apps?page=1&limit=30 获取
2. 获取智能体用户变量配置
* 通过GET接口获取:/console/api/apps/model-config/{app_id}
> 例如： /console/api/apps/model-config/efeeffdf-cb3f-4887-8eab-15e49404ffd5
3. 设置智能体用户变量值
* 通过POST接口设置变量值:/apps/model-config
1.入参：
```json
{
    "app_id":"efeeffdf-cb3f-4887-8eab-15e49404ffd5",
    "user_input_form": [
        {
            "text-input": {
                "label": "key",
                "variable": "key",
                "required": true,
                "default": "",
                "value":"4456566f5sdfds" // 这个值根据需要设置
            }
        }
    ]
}
```
只需要在每个变量里面设置value即可，其他不变
> `app_id`:为1获取的智能体id
> `user_input_form`:为2获取的智能体用户变量配置

4. 以上步骤设置完成后，即可在智能体对话页面默认设置变量的值

5. 以上接口都需要认证，token登录接口获取或者f12network获取
